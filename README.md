# PortfólioAPI
Repositório da API do meu portfólio oficial.

Para visualizar em seu computador use: 

```
git clone https://github.com/viniciuslima21/portfolioApi.git
```

## Tecnologias utilizadas
* JavaScript
* MongoDB
* Node
    * Express
    * Mongoose
    * Nodemon
    * Multer
    * Validator
    * Jsonwebtoken
    * BcryptJs
